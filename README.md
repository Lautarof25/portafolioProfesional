# Portafolio Profesional - Yésica Chevichuk

Portafolio web profesional desarrollado para展示 las habilidades, experiencia y servicios de Yésica Chevichuk, técnica en Administración con especialización en Marketing Digital y Gestión de Calidad.

## 🌐 Vista Previa

![Portafolio Preview](https://via.placeholder.com/1200x630/0a1d37/ffffff?text=Portafolio+Y%C3%A9sica+Chevichuk)

## ✨ Características

- **Diseño Responsivo**: Totalmente adaptativo para desktop, tablet y móvil
- **Animaciones Suaves**: Efectos de reveal al hacer scroll usando Intersection Observer
- **Tipografía Moderna**: Fuente Outfit de Google Fonts
- **Iconos**: Font Awesome 6.4
- **Sin Dependencias**: Solo HTML, CSS y JavaScript vanilla

## 🛠️ Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Estilos con variables CSS y clamp() para tipografía responsiva
- **JavaScript** - Interacciones nativas (Intersection Observer, mobile menu)

## 📋 Secciones

1. **Hero** - Presentación personal y llamado a la acción
2. **Sobre Mí** - Servicios ofrecidos (Gestión Administrativa, Planificación Estratégica, Marketing Digital)
3. **Experiencia** - Trayectoria profesional detallada
4. **Educación** - Formación académica y certificaciones
5. **Contacto** - Información de contacto con enlaces directos a WhatsApp y Email

## 🚀 Cómo Usar

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/portafolioYesi.git
```

2. Abre el archivo `index.html` en tu navegador

3. Para modificar el contenido:
   - Edita `index.html` para cambiar textos e imágenes
   - Modifica `styles.css` para ajustar estilos
